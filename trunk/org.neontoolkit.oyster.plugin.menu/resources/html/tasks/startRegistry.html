Starting the registry usually takes some time, depending whether the ontologies are read locally or from the internet. In any case, the process is associated to a progress bar that can be run in the background (Figure 3):

Figure3

The user can easily tell if the registry is running by checking the icon or by checking in the registry menu: when it is running, the icon is pushed and the registry menu only shows the option to stop the registry (and viceversa when it is not running).
